# Yalla Bye! 👟💥

A fast-paced, addictive "Whack-a-Mole" style game built with modern web technologies. "Yalla Bye" (Arabic for "Okay, Bye" or "Let's Go, Bye") captures the frantic energy of slapping away distractions with a trusty slipper.

## 🚀 The Story
This project was **"Vibe Coded"** into existence in **less than 24 hours**. It's a fully solo endeavor where the logic was co-authored with AI, and every sound effect and image was generated using AI tools to create a unique, cohesive aesthetic.

## 📈 Impact & Reach
- **1,500+ Players:** Rapidly gained traction with a growing community of players.
- **User-Driven Evolution:** Multiple UI/UX refinements implemented based on direct player feedback.
- **Continuous Deployment:** Deployed 3+ major iterations with new features within the first few days.

## 🛠️ Tech Stack
- **Engine:** [PixiJS v8](https://pixijs.com/) (High-performance 2D WebGL renderer)
- **Framework:** [Vite](https://vitejs.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **Styling:** Vanilla CSS (Responsive & Mobile-First)
- **Assets:** AI-Generated (Midjourney/DALL-E for images, ElevenLabs/Udion for sounds)

## ✨ Features
- **Dynamic Difficulty:** The game gets progressively harder as your score increases.
- **Responsive Design:** Optimized for both mobile "tapping" and desktop "clicking".
- **Slipper Physics:** Smooth, interactive slipper movement that follows the cursor/touch.
- **Instant Sharing:** Built-in WhatsApp sharing to brag about your high scores.

## 🛠️ Development

### Prerequisites
- Node.js (Latest LTS recommended)
- npm

### Setup
1. Clone the repository
2. Navigate to the `frontend` directory:
   ```bash
   cd frontend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
